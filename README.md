# Introduction to Artificial Intelligence

Course-related files for **Introduction to Artificial Intelligence**, Tsinghua University, Spring 2026.

This repository keeps my lecture materials and lab work for the course. Third-party review repositories and original exam archives are not copied into this repository. The `IAI_2022` review notes I used are from [zhaochenyang20/IAI_2022](https://github.com/zhaochenyang20/IAI_2022). Other exam-review materials were collected from public course-review archives and are documented here as references only.

Lab 3 was replaced by the Antwar course-competition assignment. The related project is kept separately in [Yanis01682/Ant-Game](https://github.com/Yanis01682/Ant-Game).

## Structure

- `Lectures/`: course slides and teaching materials.
- `Labs/Lab1-Pinyin-Input/`: pinyin input method lab, including source code, small test data, assignment file, and report.
- `Labs/Lab2-Sentiment-Classification/`: sentiment classification lab, including source code, dataset, word-vector file, assignment file, and report.
- `Labs/Lab3-Antwar.md`: redirect note for the Antwar replacement assignment.

The original Lab 1 package also contained large Sina News corpus files. They are not committed because several monthly files exceed GitHub's 100 MB single-file limit. Original submitted PDFs with personal identifiers are also omitted; anonymized Markdown reports are kept instead.
