# Introduction to Artificial Intelligence

Course-related files for **Introduction to Artificial Intelligence**, Tsinghua University, Spring 2026.

This repository keeps my lecture materials and my own lab implementations for the course. Third-party review repositories, original exam archives, and large datasets downloaded from course-provided cloud links are not copied into this repository.

Lab 3 was replaced by the Antwar course-competition assignment. The related project is kept separately in [Yanis01682/Ant-Game](https://github.com/Yanis01682/Ant-Game).

## Structure

- `Lectures/`: course slides and teaching materials.
- `Labs/Lab1-Pinyin-Input/`: pinyin input method lab, including source code, minimal test data, assignment PDF, and report.
- `Labs/Lab2-Sentiment-Classification/`: sentiment classification lab, including source code, dataset, word-vector file, assignment file, and report.
- `Labs/Lab3-Antwar.md`: redirect note for the Antwar replacement assignment.

Original submitted PDFs with personal identifiers are omitted; anonymized Markdown reports are kept instead. Large corpus archives referenced by assignment PDFs are documented but not committed.
