# Corpus Notes

The original lab package included two corpus folders:

- `sina_news_gbk/`: Sina News corpus from April 2016 to November 2016, encoded in GBK and formatted as JSON-like plain text.
- `SMP2020/`: Weibo data from the SMP2020 dataset.

The original Sina News corpus contains several files larger than GitHub's 100 MB single-file limit, so the raw news files are not committed to this repository. The code and report keep the preprocessing logic, and the assignment package should be used if the full raw corpus is needed.
